# 🧵 ThreadCounty

An AI-powered textile ecosystem that connects manufacturers, designers, suppliers, and buyers through intelligent sourcing, collaboration, and workflow management.

## 🚀 Overview

ThreadCounty simplifies the textile supply chain by providing a centralized platform where businesses can discover verified suppliers, manage sourcing, collaborate on projects, and make data-driven decisions using AI-powered insights.

The platform is designed to improve efficiency, transparency, and collaboration across the textile industry.

## ✨ Features

- 🤖 AI-powered supplier recommendations

- 🎨 Designer collaboration workspace

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/paridhijain5/ThreadCounty.git
```

Navigate to the project

```bash
cd ThreadCounty
```

Install dependencies

```bash
bun install
```

Run the development server

```bash
bun run dev
```
